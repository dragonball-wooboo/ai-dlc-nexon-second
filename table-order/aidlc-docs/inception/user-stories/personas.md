# 사용자 페르소나

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| **이름** | 테이블 고객 |
| **역할** | 매장 방문 고객 |
| **목표** | 빠르고 편리하게 음식을 주문하고 주문 상태를 확인 |
| **환경** | 테이블에 설치된 태블릿 (브라우저 기반) |
| **특성** | 별도 로그인 없이 즉시 사용, 터치 인터페이스 |
| **니즈** | 메뉴 탐색 → 장바구니 담기 → 주문 → 내역 확인 |
| **불편사항** | 복잡한 UI, 느린 반응, 이전 손님 주문 내역 노출 |

---

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| **이름** | 매장 관리자 |
| **역할** | 매장 운영자 / 점주 |
| **목표** | 실시간 주문 확인, 테이블 관리, 메뉴 관리 |
| **환경** | PC 또는 태블릿 브라우저 |
| **특성** | 매장 식별자 + 비밀번호 로그인, 16시간 세션 |
| **니즈** | 주문 즉시 확인 → 상태 변경 → 테이블 정리 → 메뉴 업데이트 |
| **불편사항** | 주문 누락, 테이블 혼동, 메뉴 변경 번거로움 |
